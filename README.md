# Claude Multi - лаунчер с несколькими окнами Claude

Одно окно приложения с произвольным числом независимых, постоянно
залогиненных сессий claude.ai (каждая - как отдельный профиль браузера).

## Как это работает

- При первом запуске сразу создаются 5 окон, расположенных сеткой.
- Кнопка **"+ Новое окно"** вверху добавляет ещё одну независимую сессию
  (потребуется один раз залогиниться именно в неё).
- Пока окон 5 или меньше - все видны одновременно сеткой.
- Как только окон становится больше 5 - слева появляется боковое меню
  (как список чатов в Claude): видно только одно выбранное окно на весь
  экран, остальные ждут в фоне с сохранённой сессией. Клик по пункту
  меню переключает, какое окно показано.

## Установка и запуск (один раз)

1. Установить **Node.js LTS**: https://nodejs.org
2. Распаковать папку `claude-multi-launcher` куда угодно.
3. В этой папке выполнить:

```
npm install
npm start
```

## Сборка .exe с ярлыком на рабочий стол

```
npm run dist
```

В папке `dist` появится установщик `Claude Multi.exe`. После
установки на рабочем столе будет ярлык **Claude Multi** - дальше запуск
одним кликом, все сессии сохранены.

## Где хранятся логины

Cookies/сессии каждого окна лежат на диске в `%APPDATA%\Claude Multi\Partitions\claude-session-N\`
(N - номер окна по порядку создания). Закрытие окна через ✕ убирает его
из списка, но данные на диске остаются - если когда-нибудь снова
понадобится, можно вручную удалить соответствующую подпапку, чтобы
полностью стереть тот логин.

## Полностью разлогиниться

Перед удалением приложения лучше зайти в каждое окно и сделать
Settings → Account → "Log Out" в самом Claude - это закрывает сессию
на сервере, а не только локально.
